🧱 1. Overview

Project Name

evmeri - Electric Vehicle Charging Station Locator for Ethiopia

Description

evmeri is a comprehensive mobile and web application designed to help electric vehicle owners in Ethiopia locate, navigate to, and manage charging sessions at EV charging stations. The platform provides real-time station availability, payment integration, and station management tools for both users and station owners.

Purpose & Goals

  • Primary Goal: Accelerate EV adoption in Ethiopia by providing reliable charging infrastructure information
  • User Experience: Simplify the process of finding and using EV charging stations
  • Station Management: Provide tools for charging station owners to manage their infrastructure
  • Payment Integration: Enable seamless payment processing for charging sessions
  • Localization: Support for Ethiopian users with local payment methods and languages

Key Features

  • 🗺️ Interactive Map: Real-time charging station locations using OpenStreetMap
  • 📱 Mobile App: Native Flutter application for iOS and Android
  • 🌐 Web Dashboard: Station owner management portal
  • 💳 Payment Integration: Chapa payment gateway for Ethiopian market
  • 🔍 Station Search: Find stations by location, connector type, and availability
  • ⭐ Reviews & Ratings: User feedback system for stations
  • 🚗 Navigation: Turn-by-turn directions to charging stations
  • 📊 Analytics: Usage statistics and charging history
  • 🔐 Authentication: Secure user registration and login
  • 📧 Email Verification: Account verification system
  • 🏪 Station Management: Tools for station owners to manage their infrastructure

  1. QR Code Scan: User scans station QR code
  2. Payment Initiation: Amount/kWh selection
  3. Chapa Processing: Redirect to Chapa payment gateway

  5. Charging Session: Automatic session start
  6. Session Completion: Final billing and receipt

State Management

Flutter (Provider Pattern)

// User state management
class UserProvider extends ChangeNotifier {
  User? _currentUser;
  bool _isLoading = false;

  User? get currentUser => _currentUser;
  bool get isLoading => _isLoading;

  Future<void> login(String email, String password) async {
    _isLoading = true;
    notifyListeners();

    try {
      _currentUser = await AuthService.login(email, password);
    } catch (e) {
      throw e;
    } finally {
      _isLoading = false;
      notifyListeners();
    }
  }
}

// Usage in widgets
class LoginScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Consumer<UserProvider>(
      builder: (context, userProvider, child) {
        return userProvider.isLoading
          ? CircularProgressIndicator()
          : LoginForm();
      },
    );
  }
}

React (Hooks Pattern)

// Custom hook for authentication
const useAuth = () => {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(false);

  const login = async (email, password) => {
    setLoading(true);
    try {
      const userData = await authService.login(email, password);
      setUser(userData);
      localStorage.setItem('token', userData.token);
    } catch (error) {
      throw error;
    } finally {
      setLoading(false);
    }
  };

  return { user, loading, login };
};

// Usage in components
const LoginScreen = () => {
  const { user, loading, login } = useAuth();

Performance Issues

Q: App is slow or laggy

A: Optimization steps: 1. Close other apps to free memory 2. Restart device to clear memory 3. Update app to latest version 4. Clear app cache 5. Check device storage (keep at least 1GB free)

Q: Map loading is slow

A: Improvements: 1. Use WiFi instead of mobile data when possible 2. Clear map cache in app settings 3. Reduce map quality in settings (if available) 4. Close other location-using apps
